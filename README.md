# Neon Harvest

Neon Harvest 是一个基于 Phaser、TypeScript 和 Vite 的 2D 浏览器肉鸽射击游戏。你在**黑域**里打捞霓虹碎片、用局内升级把弹幕收成一条构筑；威胁会随阶段与采样加码，首领则是技能模型拷出来的**同事复制体**与**你的终幕复写**。核心不是堆面板，而是把主武器拧成有风格的杀伤回路，并在撤离与贪场之间做取舍。详细人设见 `docs/story-world.md`。

## 模式与撤离

- **战役**：无尽阶段推进；**每个新阶段**会弹出可关闭的**黑域打字机叙事**（冻结战斗直至关闭）；单阶段目标量高于清剿；完成第 12 阶段主线后可选择继续清剿或撤离结算；约 8 分钟后撤离信标解锁。
- **清剿**：与战役相同的阶段与撤离规则，但**不播放阶段叙事**，节奏更偏纯清场。
- 桌面 HUD 与地图上会强化撤离信标提示（高亮区、屏外方向标等）。

## 当前内容

- 无限地图、随机障碍物、区块式自动生成
- 视野系统：中心明亮、边缘渐暗
- 多把差异化武器与角色技能
- 普通敌人、精英敌人、随机与结点 **复制体**（同事副本 / 终幕复写）
- 升级卡构筑、机库永久升级、本地排行榜
- 桌面端键鼠操作与手机端触控操作

## 本地运行

```bash
npm install
npm run dev
```

默认开发地址通常是 `http://localhost:4173`。

## 生产构建

```bash
npm run build
```

## 操作说明

### 桌面端

- `WASD`：移动
- `鼠标`：瞄准和射击
- `Shift` 或 `Space`：冲刺
- `E`：在撤离点内交互撤离
- `Esc`：暂停

### 手机端

- 左下虚拟摇杆：移动
- 右下瞄准盘：瞄准并持续射击
- 右侧按钮：冲刺 / 交互 / 暂停

## 排行榜

- 排行榜为本地排行榜，保存在浏览器 `localStorage`
- 每局结算后自动记录成绩
- 默认保留前 10 名，按回收积分排序

## 目录结构

```text
src/
  game/
    content/        武器、技能、升级等静态数据
    input/          键鼠与触控输入
    simulation/     核心战斗与结算逻辑
    storage/        本地存档
  phaser/
    scenes/         Phaser 场景与渲染
    view/           贴图与视觉辅助
  ui/
    appShell.ts     DOM HUD、主界面、机库、排行榜
  styles.css        全局样式
```

## 存档说明

- 游戏进度、机库升级、已解锁武器、最近战绩、排行榜都保存在当前浏览器本地
- 更换浏览器或清除站点数据后，存档会丢失
